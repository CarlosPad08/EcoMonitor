# Fuentes Poppins

Para usar la fuente Poppins, descarga los archivos desde Google Fonts:

1. Ve a https://fonts.google.com/specimen/Poppins
2. Descarga los archivos:
   - Poppins-Regular.ttf
   - Poppins-Medium.ttf
   - Poppins-SemiBold.ttf
   - Poppins-Bold.ttf

3. Colócalos en esta carpeta

O usa el comando:
```bash
# Descargar Poppins desde Google Fonts
curl -o Poppins-Regular.ttf "https://fonts.gstatic.com/s/poppins/v20/pxiEyp8kv8JHgFVrJJfecg.woff2"
```

**Nota**: Por ahora la app funcionará con las fuentes del sistema hasta que agregues los archivos de Poppins.
